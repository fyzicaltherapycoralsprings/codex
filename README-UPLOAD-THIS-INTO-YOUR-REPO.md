# Upload pack for fyzicaltherapycoralsprings/codex

This pack contains the core documentation and instruction files to add into the existing GitHub repo.

## What to do
1. Extract this zip.
2. Open the existing GitHub repo in the browser.
3. Use **Add file → Upload files**.
4. Drag the extracted folder contents into the browser window.
5. Commit the changes.

## Included
- `AGENTS.md`
- `.codex/config.toml`
- core docs in `docs/`
- runbooks in `runbooks/`
- official source map in `sources/`
- durable record starters in `records/`
- templates in `templates/`
